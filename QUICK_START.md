# 🚀 快速开始指南

5分钟内开始使用LeetCode刷题复习助手！

## 第一步：加载扩展到Chrome

1. 打开Chrome浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 打开右上角的**"开发者模式"**开关
4. 点击**"加载已解压的扩展程序"**
5. 选择本项目文件夹（`leetcode-spaced-repetition`）
6. 完成！扩展图标会出现在Chrome工具栏

## 第二步：添加第一道题

1. 访问任意LeetCode题目，例如：
   - 国际版：https://leetcode.com/problems/two-sum/
   - 中国版：https://leetcode.cn/problems/two-sum/

2. 等待页面加载完成

3. 你会看到页面右侧出现一个**紫色渐变按钮**："加入复习"

4. 点击按钮，题目就添加成功了！🎉

5. 系统会自动安排6次复习：
   - 1天后
   - 3天后
   - 7天后
   - 14天后
   - 30天后
   - 60天后

## 第三步：查看复习计划

1. 点击Chrome工具栏中的扩展图标

2. 在弹出窗口中你会看到：
   - **今日复习**：今天需要复习的题目
   - **全部题目**：所有已添加的题目
   - **设置**：配置选项

## 每日使用流程

### 早上/晚上查看今日任务

1. 打开扩展弹出窗口
2. 查看"今日复习"标签页
3. 点击"打开题目"重新做一遍
4. 完成后点击"完成复习"按钮

### 刷新题时添加到计划

1. 在LeetCode上做新题
2. 做完后点击页面上的"加入复习"按钮
3. 题目会自动加入复习计划

## 可选：连接Google Calendar

如果你想把复习计划同步到Google Calendar：

1. 查看 `SETUP_GUIDE.md` 中的详细配置步骤
2. 配置Google Calendar API
3. 在扩展的"设置"标签页点击"连接Google Calendar"

**不配置也完全可以使用！** 扩展会通过Chrome通知提醒你。

## 核心功能一览

| 功能 | 说明 |

## 使用小技巧

1. **每天固定时间查看**：建议每天晚上8点查看今日复习任务
2. **及时标记完成**：做完题后立即点击"完成复习"，保持记录准确
3. **定期导出数据**：在设置中定期导出数据作为备份
4. **删除已掌握的题**：对于已经完全掌握的题目，可以删除以保持列表清爽

## 遗忘曲线说明

本扩展基于**艾宾浩斯遗忘曲线**理论：

- 人们在学习新知识后，遗忘的速度最初很快，之后逐渐减慢
- 通过在关键时间点复习，可以有效对抗遗忘
- 6次复习后，知识基本会转入长期记忆

复习间隔：**1天 → 3天 → 7天 → 14天 → 30天 → 60天**

## 常见问题

### Q: 浮动按钮看不到？
A: 确保你在题目详情页（URL包含`/problems/`），然后刷新页面。

### Q: 如何改变复习时间？
A: 在"设置"标签页中可以修改每日提醒时间。

### Q: 数据存在哪里？
A: 数据存储在Chrome本地存储中，只在你的电脑上，不会上传。

### Q: 如何在多台电脑同步？
A: 使用"导出数据"和"导入数据"功能，或配置Google Calendar。
