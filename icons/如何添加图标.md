# 如何添加图标

## 方法1：使用自带的图标生成器（最简单）⭐

1. 用浏览器打开项目根目录的 `generate-icons.html` 文件
2. 点击"下载全部"按钮
3. 将下载的三个PNG文件移动到这个 `icons/` 文件夹
4. 在 `manifest.json` 中添加图标配置：

```json
  "action": {
    "default_popup": "popup.html",
    "default_icon": {
      "16": "icons/icon16.png",
      "48": "icons/icon48.png",
      "128": "icons/icon128.png"
    }
  },
  "icons": {
    "16": "icons/icon16.png",
    "48": "icons/icon48.png",
    "128": "icons/icon128.png"
  },
```

5. 在 `chrome://extensions/` 重新加载扩展

## 方法2：使用在线工具

访问这些网站快速生成图标：
- https://favicon.io/
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

## 方法3：暂时不添加（当前方案）

扩展已经配置为可以在没有图标的情况下运行，只是会显示Chrome默认图标。
你可以随时添加图标，添加后重新加载扩展即可。

## 所需文件

在这个文件夹中需要三个PNG图标：
- `icon16.png` - 16×16像素
- `icon48.png` - 48×48像素
- `icon128.png` - 128×128像素
