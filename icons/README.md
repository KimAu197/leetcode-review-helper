# 图标文件说明

请在此文件夹中添加以下三个PNG图标：

- `icon16.png` - 16x16像素
- `icon48.png` - 48x48像素  
- `icon128.png` - 128x128像素

## 如何创建图标

### 方法1: 使用在线工具
1. 访问 https://favicon.io/ 或 https://www.favicon-generator.org/
2. 上传图片或创建文字图标
3. 下载生成的图标
4. 重命名为对应尺寸并放入本文件夹

### 方法2: 使用Figma/Photoshop
1. 创建新画布，设置对应尺寸
2. 设计图标（建议使用紫色渐变主题，与扩展风格一致）
3. 导出为PNG格式
4. 保存到本文件夹

### 方法3: 使用AI生成
你可以让AI（如DALL-E、Midjourney）生成一个书籍+记忆曲线的图标设计

## 设计建议

- **主题**: 书籍📚 + 脑部记忆💡 + 曲线📈
- **颜色**: 使用紫色渐变 (#667eea -> #764ba2)
- **风格**: 现代、简洁、扁平化
- **对比度**: 确保在浅色和深色背景下都清晰可见

## 临时方案

如果暂时没有图标，扩展仍然可以正常运行，只是会显示Chrome的默认图标。
你可以稍后再添加图标，添加后重新加载扩展即可生效。
